## Install

### Prerequisites

- [Node.js]() v12.0+
- [Yarn]() v1.22.0+

### Install & development

```sh
# Clone repository
git clone https://github.com/farhan2077/symbols-search

# Change directory
cd symbols-search

# Install dependencies
yarn

# Start dev server
yarn dev
```

Open `http://localhost:3000` in your browser.
